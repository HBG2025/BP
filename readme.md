# BP - Basic Programming

Repositorio de materiales de apoyo para cursos introductorios de programación.

Este repositorio reúne documentos, guías y prácticas orientadas al aprendizaje progresivo de conceptos fundamentales de programación, especialmente en lenguaje C, C++ y entornos aplicados como Arduino.

## Materiales disponibles

### Punteros y memoria en C

Carpeta: [`punteros-c`](./punteros-c)

Esta sección contiene materiales para estudiar, paso a paso, la relación entre memoria, variables, direcciones y punteros en lenguaje C.

Incluye:

- **Arquitectura general y memoria del programa**  
  Introducción a la memoria, direcciones, programa almacenado, programa en ejecución, variables locales y funciones.

- **Punteros en C**  
  Explicación de valor, dirección, operadores `&` y `*`, paso de direcciones a funciones, modificación indirecta de variables y punteros como canales de salida.

- **Práctica Arduino con funciones, punteros y arreglos**  
  Actividad aplicada con potenciómetro, LED, funciones, punteros, arreglos y control básico de una salida.

## Propósito

El objetivo de estos materiales es apoyar el aprendizaje de programación desde una perspectiva clara, práctica y progresiva. Los documentos buscan relacionar la teoría con ejemplos completos y actividades que permitan ejecutar, observar y modificar código.

## Uso sugerido

Los materiales pueden ser utilizados como:

- guías de estudio;
- apoyo para clases de programación;
- material de laboratorio;
- referencia para prácticas en C y Arduino;
- base para talleres y ejercicios.
